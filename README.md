# system-monitor
a widget showing the characteristics of your computer

Hello everyone, this is a project I created in python.

this project displays data from your processor, video card, RAM, and SSD
# how to install it:
download the file installer.bat

run the installer

if you don't have python installed, it will also install the dependencies

also download the source file

It is important to download the source code before using the installer and place it in the same location as the installer

after the build, the project will start

Enjoy using it
